= 创建 Snapshot 副本
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
在网络连接存储（ NAS ）中使用存储区域网络（ Storage Area Network ， SAN ）和 NFS 卷时，您可以使用一个命令为多个文件系统，主机卷， LUN 或磁盘组创建 Snapshot 副本。

.步骤
. 创建 Snapshot 副本：
+
` * SnapDrive snap create -fs _file_spec_ -snapname _snapshot_copy_name_*`
