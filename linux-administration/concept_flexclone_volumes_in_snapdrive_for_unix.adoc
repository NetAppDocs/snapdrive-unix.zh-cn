= SnapDrive for UNIX 中的 FlexClone 卷
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
SnapDrive for UNIX 支持基于 Data ONTAP FlexClone 技术的 FlexClone 卷。FlexClone 卷是父灵活卷的时间点副本。FlexClone 卷比 LUN 克隆速度更快，它们可以让您灵活地创建多个数据副本。
