= 传输协议连接失败时， SnapDrive 命令失败
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
如果主机和存储系统之间的 iSCSI 或 FC 连接失败，或者存储系统上的 iSCSI 或 FC 出现故障，则 SnapDrive 命令将失败。



== 临时解决策

要成功使用 SnapDrive for UNIX 命令，必须先还原 iSCSI 或 FC 服务。
