= 显示存储系统上的 Snapshot 副本
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用 SnapDrive 命令显示存储系统上的 Snapshot 副本。

要显示有关驻留在存储系统上的 Snapshot 副本的信息，请使用以下语法：

SnapDrive snap show -filer filername [filername...] 【 -verbose 】``
