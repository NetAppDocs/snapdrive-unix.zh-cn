= 确定可以创建的 LUN 数量
:icons: font
:imagesdir: ../media/


[role="lead"]
通过 SnapDrive for UNIX ，您可以在不超过限制的情况下确定可以在主机上创建多少个 LUN 。

您可以使用 SnapDrive config check lun` 命令来确定此值。

在 Linux 主机上，此命令会检查现有的 /dev/sg 文件，以确定未使用的文件数。
