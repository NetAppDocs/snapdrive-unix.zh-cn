= 使用向导执行的操作
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用不同的向导高效地创建存储实体并操作 SnapDrive for UNIX 。

以下 SnapDrive for UNIX 命令具有一个等效的向导来执行操作：

* `s存储创建`
* `s存储删除`
* `snap restore`
* `snap connect`
* `snap disconnect`
