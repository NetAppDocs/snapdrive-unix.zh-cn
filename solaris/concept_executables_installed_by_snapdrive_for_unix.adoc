= SnapDrive for UNIX 安装的可执行文件
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
SnapDrive for UNIX 会为 Solaris 操作系统安装可执行文件。

这些可执行文件安装在以下位置：

* ` /opt/NTAPsnapdrive/bin/SnapDrive`
* ` /opt/NTAPsnapdrive/bin/snapdrived`
