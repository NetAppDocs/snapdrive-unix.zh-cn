= 还原存储系统或存储系统 HA 对上的单个存储实体
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以还原包含单个存储实体的 Snapshot 副本。

您可以还原包含驻留在存储系统或存储系统 HA 对上的单个存储实体的 Snapshot 副本。目标存储系统上的卷名称必须与源存储系统上的卷名称匹配。
