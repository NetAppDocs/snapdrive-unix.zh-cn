= 安装 Solaris Host Utilities
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
如果您的配置需要 Solaris Host Utilities ，则必须安装它们并确保系统设置正确。

您应使用 Solaris Host Utilities 附带的文档。其中包含有关卷管理器，多路径以及在安装 SnapDrive for UNIX 之前必须设置的其他功能的信息。
