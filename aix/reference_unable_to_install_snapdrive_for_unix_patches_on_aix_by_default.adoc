= 默认情况下，无法在 AIX 上安装 SnapDrive for UNIX 修补程序
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
默认情况下，不能使用系统管理界面工具（ SMIT ）在 AIX 上安装 SnapDrive for UNIX 修补程序版本。



== 临时解决策

安装适用于 AIX 的 SnapDrive for UNIX 修补程序版本时，请确保在 SMIT 中选择 `overwrite` 选项。
