= SnapDrive for UNIX 安装的可执行文件
:icons: font
:imagesdir: ../media/


[role="lead"]
SnapDrive for UNIX 将安装适用于 AIX 操作系统的可执行文件。

这些可执行文件安装在 ` /opt/NetApp/SnapDrive/bin/SnapDrive` 中。
