= SnapDrive for UNIX 中的多路径
:icons: font
:imagesdir: ../media/


[role="lead"]
SnapDrive for UNIX 支持多路径。SnapDrive for UNIX 不支持对使用 `m节` 和 / 或 `rendev` 命令创建的设备使用别名。
